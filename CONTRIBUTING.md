| Estructura de commits| 
| Tipo | Propósito | Ejemplo |
| :--- | :--- | :--- |
| `feat` | Nueva característica (RF). | `feat(login): Implementar validación de credenciales.` |
| `fix` | Corrección de error (Defecto/Issue). | `fix(database): Corregir error de tipo en columna ID.` |
| `docs` | Cambios en la documentación. | `docs(readme): Actualizar sección de dependencias.` |
| `refactor` | Reestructuración de código. | `refactor(auth): Optimizar manejo de tokens.` |
| `test` | Adición o corrección de pruebas. | `test(api): Agregar casos de prueba para el GET.` |
